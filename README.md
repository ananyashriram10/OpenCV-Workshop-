
# OpenCV Workshop 
The workshop introduces fundamental computer vision concepts using **OpenCV with Python**, focusing on practical implementation and real time examples.

---

## Module 1: Getting Started
- Importing images, videos, and files
- Understanding video formats and **FOURCC codes**
- Accessing and using the webcam
- Basic arithmetic operations on images
- Drawing shapes and text on images

---

## Module 2: Image Processing Basics
- Color spaces: **RGB, HSV, Grayscale**
- Thresholding techniques
- Blurring and filtering
- Morphological transformations:
  - Erosion
  - Dilation
  - Opening
  - Closing

---

## Module 3: Feature Detection & Analysis
- Image gradients:
  - Sobel
  - Laplacian
- Edge detection
- Image blending using **image pyramids**
- Contour detection and analysis
- Masking techniques
